# Fine3300-Stock_Chart
## Introduction

This project retrieves IVV stock data using **yfinance** and visualizes it with **mplfinance** to generate a candlestick chart for January 2025. The development of this project was guided by online resources that addressed customization of mplfinance plots and solutions for common plotting issues. For more details on these approaches, please refer to:

- [Customizing mplfinance plot](https://stackoverflow.com/questions/60599812/how-can-i-customize-mplfinance-plot)
- [Handling TypeError: only length-1 arrays can be converted to Python scalars while plotting](https://stackoverflow.com/questions/36680402/typeerror-only-length-1-arrays-can-be-converted-to-python-scalars-while-plot-sh)
- [Error: Data for column open must be all float or int](https://stackoverflow.com/questions/79133206/mplfinance-error-data-for-column-open-must-be-all-float-or-int)
- [yfinance Documentation](https://yfinance-python.org/index.html)
- [mplfinance GitHub Repository](https://github.com/matplotlib/mplfinance/tree/master?tab=readme-ov-file)
